# Collisions

The following helper functions are provided for 2D area checking:

- [`point_in_rectangle(px, py, x1, y1, x2, y2)`](https://manual.yoyogames.com/GameMaker_Language/GML_Reference/Movement_And_Collisions/Collisions/point_in_rectangle.htm)
- [`point_in_triangle(px, py, x1, y1, x2, y2, x3, y3)`](https://manual.yoyogames.com/GameMaker_Language/GML_Reference/Movement_And_Collisions/Collisions/point_in_rectangle.htm)
- [`point_in_circle(px, py, x1, y1, rad)`](https://manual.yoyogames.com/GameMaker_Language/GML_Reference/Movement_And_Collisions/Collisions/point_in_circle.htm)
- [`rectangle_in_rectangle(sx1, sy1, sx2, sy2, dx1, dy1, dx2, dy2)`](https://manual.yoyogames.com/GameMaker_Language/GML_Reference/Movement_And_Collisions/Collisions/rectangle_in_rectangle.htm)
- [`rectangle_in_triangle(sx1, sy1, sx2, sy2, dx1, dy1, dx2, dy2, dx3, dy3)`](https://manual.yoyogames.com/GameMaker_Language/GML_Reference/Movement_And_Collisions/Collisions/rectangle_in_triangle.htm)
- [`rectangle_in_circle(sx1, sy1, sx2, sy2, x, y, rad)`](https://manual.yoyogames.com/GameMaker_Language/GML_Reference/Movement_And_Collisions/Collisions/rectangle_in_circle.htm)
